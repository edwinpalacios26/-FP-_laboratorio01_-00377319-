git add.
Agrega todos los archivos al repositorio.

git commit -m "un Commit" comentar los cambios que hemos hecho.

git clone 
Se usa para copiar un repositorio y trabajar con él nuestra máquina.

git push -u origin master

Guardamos nuestros cambios en la nube y agregamos a la rama principal.
